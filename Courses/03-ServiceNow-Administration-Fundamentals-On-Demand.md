# ServiceNow Administration Fundamentals On Demand

## Course Overview

ServiceNow Administration Fundamentals On Demand is the official administrator training course offered by ServiceNow University.

This course builds upon the introductory concepts learned in Welcome to ServiceNow and prepares learners for real-world platform administration.

It covers instance administration, security, platform configuration, scripting fundamentals, automation, and best practices.

---

## Learning Objectives

After completing this course, learners should be able to:

- Configure ServiceNow instances.
- Manage Users, Groups and Roles.
- Work with Tables and Fields.
- Configure Forms and Lists.
- Manage Access Controls (ACLs).
- Configure Notifications.
- Create Reports.
- Manage Applications.
- Understand Data Management.
- Learn platform administration best practices.
