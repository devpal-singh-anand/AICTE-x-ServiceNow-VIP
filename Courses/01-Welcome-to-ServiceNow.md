# Welcome to ServiceNow

## Course Overview

**Welcome to ServiceNow** is the official introductory course offered through ServiceNow University.

It provides a beginner-friendly introduction to the ServiceNow platform, covering the user interface, navigation, records, forms, lists, and core platform concepts. The course is designed for learners who are completely new to ServiceNow and serves as the recommended starting point before pursuing more advanced administration or development topics.

---

## Learning Objectives

By completing this course, learners should be able to:

- Understand what ServiceNow is.
- Navigate the ServiceNow interface.
- Work with Lists and Forms.
- Understand Records and Tables.
- Learn basic platform navigation.
- Explore common ServiceNow applications.
- Become familiar with the ServiceNow ecosystem.

---

## Topics Covered

- ServiceNow Platform Introduction
- User Interface
- Navigation
- Lists
- Forms
- Records
- Tables
- Applications
- Modules
- Basic Platform Features
